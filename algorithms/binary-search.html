Understanding the Binary Search Algorithm

This is also know as "Diving and Conquering"

Instead of doing a simple search and searching number by number, a Binary Search allows us to divide out search in half each time we look for a target value

this makes the search overall faster and much more efficent

Important! Binary Search only works with sorted arrays

We define a middleIndex (or a midpoint) from the elements in the middle of the array (this is the "Divide") to see if our target values is greater than or less than the middleIndex or midpoint

Depending on if the target value is greater than or less than the middleIndex, we can remove the left or right of our array from our search (This is the "Conquer")
